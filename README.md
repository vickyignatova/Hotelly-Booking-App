# Hotelly Booking App

Hotel booking app project designed to improve the online reservation experience through immersive UI/UX design and virtual hotel exploration.

## Project Overview

When booking a hotel online, users usually rely only on photos and reviews, which do not always reflect the real atmosphere of the place. Because of this, it can be difficult to know whether a hotel will truly match user expectations.

Hotelly solves this problem by introducing a **VR mode** that allows users to explore the hotel virtually before booking. This gives them a better sense of the atmosphere, layout, and overall experience, helping them make a more informed and confident decision.

## Main Feature

**VR Mode**  
A virtual exploration feature that lets users experience the hotel space before making a reservation.
